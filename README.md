# Bioinformatics Pipeline

## Project Description
This project implements a simple bioinformatics pipeline for processing sequencing data.
It includes modules for quality control and validation of input FASTQ files to ensure reliable downstream analysis.

## Dependencies
The following packages are required to run this project:

- Python 3.8 or higher
- numpy

## Installation Instructions
1. Clone the repository:
   git clone https://github.com/wonderfianu/bioinfo-pipeline

2. Navigate into the project directory:
   cd bioinfo-pipeline

3. Install dependencies:
   pip install numpy

## Execution Instructions
To run the pipeline:

```bash
python src/pipeline.py
